A sleek, interactive web project built with modern Vanilla Web Technologies.

 Project Overview
This project is a lightweight, responsive web application focusing on smooth user experiences. It demonstrates the power of core web technologies without the need for heavy frameworks.

 🛠 Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom styling and Keyframe animations.
- **Vanilla JavaScript:** DOM manipulation and interactive logic.

 Key Features
- Interactive UI: Dynamic elements powered by Vanilla JS.
- Custom Animations: Smooth transitions using pure CSS.
- Responsive Design: Optimized for all screen sizes.
